# Pixer Day 1 Frontend

Frontend foundation and project setup for the Pixer marketplace, based only on
the Day 1 task document.

## Day 1 Scope From `Day.pdf`

- Understand marketplace flow, modules, and user roles.
- Create React project setup with Vite and npm.
- Create a clean folder structure.
- Configure Bootstrap 5.
- Build responsive homepage sections.
- Create responsive UI cards.
- Create reusable Navbar and Footer components.
- Configure routes with React Router DOM.
- Build a responsive landing page.
- Prepare local Git workflow basics.

## Commands

```bash
npm install
npm run dev
npm run build
```
